## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hberkaykuran/sts-tierlist.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Execute run command and access the site http://localhost:3000
   ```sh
   yarn dev
   ```
